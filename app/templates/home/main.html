Synopsis

This is a Flask application holding the API for my Participant-Gauntlent Modeler. The links in the navigation will direct you to the pages and necessary instructions for using the API.

  • Populate Database:
    A list of forms which, upon use, will populate the database with all the relevant information.
    These forms are meant to be used once, as they activate scripts which add records to a database.
    If used multiple times, they will add duplicate information to the database.
    Link scripts will be commented out after their first use, but are left there in case the database becomes FUBAR
  • API:
    A page with API instructions, sample queries, and a form to query data directly.
